# Bayesian Optimization with Discrete Variables

This repository contains the implementation of the method DiscreteBO proposed in the paper 'Bayesian Optimization with Discrete Variables', AI2019.

## Prerequisites

- Python 3.6
- Numpy 1.18
- Scipy 1.4
- Gpy 1.9.9
- Numdifftools 0.9.39
